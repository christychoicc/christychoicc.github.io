## MSc Urban Spatial Science Coursework

### Spatial Analysis - CASA0005 Geographic Information Systems and Science

#### New York Evictions
This is a spatial hotspot analysis examining where eviction activities cluster across New York in 2020 using evictions data provided by NYC Open Data. The analysis applied point pattern analysis methods and spatial autocorrelation methods to identify statistically significant hotspot and coldspot areas, discussing areas to be prioritiesed for policy interventions.

View the full report (HTML): [New York Evictions](https://christychoicc.github.io/coursework/New-York-Evictions.html)

#### TfL Vision Zero 
This is a spatial hotspot analysis examining where deaths and serious injuries cluster across London in 2024 using casualty data provided by TfL. The analysis applied point pattern analysis methods and spatial autocorrelation methods to identify statistically significant hotspot and coldspot areas and discuss policy implications for Vision Zero.

View the full report (HTML): [TfL Vision Zero](https://christychoicc.github.io/coursework/TfL-Vision-Zero.html)
